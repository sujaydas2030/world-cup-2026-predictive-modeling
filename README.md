# World Cup 2026 Predictive Modeling

This project uses a PySpark pipeline to evaluate international football teams based on historical World Cup baselines (1990–2022) to project tournament semifinalists.

### 📊 Methodology
* **Baseline Metrics:** Median goal difference, goals conceded, and clean sheets of past semifinalists.
* **Tie-Breaker Logic:** Highest Avg Goal Difference -> Lowest Avg Goals Conceded -> Highest Clean-Sheet Count.

### 🔮 Predicted Top 4
* Japan & Spain (Dominant goal margins)
* Argentina & Portugal (Defensive resilience)

*Note: Built using PySpark to demonstrate scalable data engineering pipeline design.*
